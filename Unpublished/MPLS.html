MPLS

Is a service provider technology but is also used by enterprises. It is also used to replace dedicated serial lines to interconnect separate offices.

MPLS is globally enabled. MPLS is supported on almost all Cisco routers. By default, MPLS traffic follows the same path as regular IP traffic would. By default, all IGP routes will have a label assigned.

MPLS consists of:

  • MPLS data plane: how MPLS traffic is forwarded in, out and throughout the MPLS network
  • Control plane: deals with how the labels are assigned and exchanged in the MPLS network.

MPLS Data Plane

Terms used in MPLS

Assumption is that the MPLS control-plane is fully functional.

  • Label Switching Routers (LSRs): all participating routers in MPLS network.
  • Edge LSR: The LSRs at the edge of the network where traffic enters and exits the MPLS network are referred to as edge routers.
  • Ingress Edge LSR: The edge LSR where IP packets enter the MPLS network.

When an incoming IP packet enters the ingress LSR, a lookup is done in the Label Forwarding Information Base(LFIB). The LFIB contains information on how the label gets switched.
