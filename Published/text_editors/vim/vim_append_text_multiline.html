To add text at the end of multiple lines in Vim, you can use Visual Block Mode or a substitution command . Below are two common methods to achieve this:

Method 1: Using Visual Block Mode

This method is ideal when you want to append text to specific lines.

  1. Enter Visual Block Mode :

    • Move your cursor to the first line where you want to add text.

    • Press Ctrl-v to enter Visual Block Mode.

  2. Select the Lines :

    • Use the arrow keys (or j/k) to select the lines where you want to append text. This will create a vertical block selection.

  3. Move to the End of Each Line :

    • Press $ to extend the selection to the end of each line.

  4. Append Text :

    • Press A (shift + a) to start appending text at the end of the selected lines.

    • Type the text you want to add, and it will be added to all selected lines.

  5. Exit Insert Mode :

    • Press Esc, and the text will be appended to all selected lines.

Method 2: Using a Substitution Command

This method is useful when you want to append text to all lines in a file or a range of lines.

  1. Use the :substitute Command :

    • To append text to all lines in the file, type: :%s/$/your_text/

  • Here:

  • % specifies all lines in the file.

  • $ matches the end of each line.

  • your_text is the text you want to append.

  • To append text to a specific range of lines (e.g., lines 5 to 10), type: :5,10s/$/your_text/

  • Confirm Changes :

  • Press Enter, and the text will be appended to the specified lines.
